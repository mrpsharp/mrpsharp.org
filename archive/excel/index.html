Excel for sixth form physicists

Excel is an extremely capable program and full of opportunities for sixth form physicists to present and analyse data, fit mathematical models to existing data and create dynamic models of processes which they are not able to analyse mathematically

Simple Table and Graph

Learn how to use Excel to produce a properly formatted table an graph. The graph will include a linear best fit line.

Error bars

Learn how to add error bars to graphs along with custom best fit lines.
